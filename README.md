# RN App Sample TS

## Integrations

* TypeScript
* React Navigation
* Babel Module Resolver

## Change App Name

Rename the app with [React Native Rename](https://www.npmjs.com/package/react-native-rename)

`npm run rename <newName>`

## Init project

1- Install node_modules

`yarn`

2- Install Pods (iOS)

`npx pod-install`
